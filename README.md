# Qwrapper

QWrapper is an abstract API which makes working with different message queues simple.

## Installation

Add this line to your application's Gemfile:

    gem 'qwrapper'

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install qwrapper
